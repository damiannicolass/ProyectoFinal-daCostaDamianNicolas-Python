Aplicación Web para Gestionar Clientes

Esta aplicación web ha sido desarrollada para ofrecer una solución completa y eficiente en la gestión de negocios de seguros.

Características Principales:

  • Gestión de Clientes y Empleados: Visualiza, carga y busca información sobre clientes y empleados.
  • Administración de Vehículos: Gestiona datos sobre vehículos asegurados.
  • Gestión de Pólizas: Registra y controla las pólizas de seguros, marcando su estado (vigente, anulada o vencida).
  • Tareas y Siniestros Pendientes: Agrega, visualiza y administra tareas y siniestros con un sistema de comentarios para seguimiento.
  • Inspecciones de Vehículos: Crea y lista inspecciones de vehículos, permitiendo a los asegurados enviar fotos desde sus dispositivos móviles para una actualización en tiempo real.

Objetivo del Proyecto:

Este proyecto tiene como objetivo proporcionar una herramienta completa y fácil de usar para la gestión de seguros. La aplicación facilita la gestión diaria de los productores asesores de seguros.
